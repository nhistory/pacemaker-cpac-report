# CPAC 모델 vs 빌딩업 모델 비교 및 '페이스메이커' 적용 분석

---

## 📊 한눈에 보는 비교표

| 비교 항목   | CPAC 모델 (이원화 구조)                                                          | 빌딩업(Building Up) 모델 (단일형 사회적 기업)                                  |
| ----------- | -------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 핵심 구조   | 영리(기존) + 비영리(신규) = 2개 법인                                             | 비영리 법인 하나가 비즈니스도 수행 = 1개 법인                                  |
| 운영 방식   | 따로 또 같이 (Two-track) - 수익 사업과 공익 사업을 분리하여 운영                 | 올인 (Integrated) - 비즈니스 자체가 곧 공익 활동 (예: 취약계층 고용)           |
| 주 수입원   | 영리: 비즈니스 매출 (B2B/B2C) / 비영리: 정부 펀딩, 기부금, 그랜트                | 매출(약 60%): 용역 계약, 물품 판매 / 보조금(약 40%): 훈련비 지원, 정부 그랜트  |
| 인력 운영   | 영리 직원이 비영리 업무를 겸하거나 지원 (초기), 비영리는 최소 인력으로 운영 가능 | 자체적으로 대규모 인력을 직접 고용 (비즈니스를 돌려야 하므로 인건비 비중 높음) |
| 장점        | ✅ 안전성 & 유연성 - 기존 사업을 지키면서 펀딩 채널만 추가                       | ✅ 자생력 & 확장성 - 펀딩 없이도 생존 가능, 사회적 임팩트가 큼                 |
| 단점        | ⚠️ 이해상충 관리 복잡 - 두 법인 간 자금/인력 거래 시 CRA 규제 주의               | ⚠️ 초기 진입 장벽 높음 - 확실한 수익 모델(기술, 수주 능력) 없으면 실패         |
| 적합한 상황 | 이미 잘되는 영리 사업이 있고, 공공 펀딩 통로가 추가로 필요한 경우                | 맨땅에서 시작하거나, 비즈니스 모델 자체가 사회적 가치와 직결된 경우            |

---

## 🎯 왜 '페이스메이커'에게는 CPAC 모델이 유리한가?

YMCA, KCWA, 영사관 등의 조언을 종합해 볼 때, 페이스메이커에게는 CPAC 모델이 압도적으로 유리합니다.

### 1️⃣ 기존 영리 사업의 '자산과 수익'을 지킬 수 있습니다

| 빌딩업 모델로 가면                                       | CPAC 모델로 가면                                    |
| -------------------------------------------------------- | --------------------------------------------------- |
| 기존 회사를 비영리 법인으로 전환하거나 흡수시켜야 함     | 페이스메이커(영리)에서 번 돈은 정당한 수익으로 유지 |
| 소유권(지분)을 포기하고 이익을 개인적으로 가져갈 수 없음 | 새로 만든 비영리 법인을 통해 공익 사업 가능         |
|                                                          | → '내 것'을 지키면서 '좋은 일'도 할 수 있음         |

### 2️⃣ 파트너 기관(YMCA 등)이 원하는 것은 '법적 명분'뿐입니다

- 파트너 기관들은 행정적으로 문제가 없는 '비영리 사업자등록번호(BN)'만 필요
- 조직 전체를 뜯어고칠 필요 없이, 계약을 맺을 수 있는 '비영리 파트너 법인(Vessel)' 하나만 가볍게 설립하는 것이 효율적

### 3️⃣ 리스크 분산 (Safety Net)이 가능합니다

| 빌딩업 모델 (단일 법인)                                | CPAC 모델                                        |
| ------------------------------------------------------ | ------------------------------------------------ |
| 펀딩이 끊기거나 비즈니스가 어려워지면 조직 전체가 위험 | 비영리 쪽 펀딩이 안 되면 → 영리 쪽 매출로 보전   |
|                                                        | 영리가 주춤하면 → 비영리 쪽 그랜트로 인건비 보전 |
|                                                        | → 상호 보완 가능                                 |

### 4️⃣ '커리어 코칭'이라는 업의 특성 때문입니다

- 코칭/교육은 무형의 서비스로, 전면 무료화하거나 비영리로만 돌리기에는 수익 구조가 취약
- 고가 유료 코칭(영리) + 보급형 무료 코칭(비영리-정부지원)을 나누어 운영하는 CPAC 식 '투 트랙 전략'이 시장 논리에 훨씬 부합

## 🔀 CPAC 모델의 '투 트랙 전략' 예시

### Track A: 비영리 법인 (Paceup Career Society)

> 💰 "정부가 돈을 낸다"

| 항목          | 내용                                                                                          |
| ------------- | --------------------------------------------------------------------------------------------- |
| 상품명 (예시) | "IT/Design Bridging Program for Newcomers"                                                    |
| 타겟          | 구직 중인 영주권자, 난민, 또는 워크퍼밋 소지자 (정부 지원 요건 충족자)                        |
| 내용          | 현재 진행 중인 6~8주 코스를 활용하되, '취업 역량 강화(Employment Services)'라는 명분으로 포장 |
| 수익 구조     | 수강생에게는 무료로 제공 / 정부나 재단(YMCA, 영사관 등)으로부터 운영비와 강사료를 지원받음    |

⭐ 핵심 포인트:

> 정부 지원금 예산에 멘토(디렉터, 시니어 개발자)의 강사료(Instructor Fee)를 정당한 시장 가격(시간당 $100~$150 등)으로 책정하여, 멘토들에게 '열정 페이' 없이 정당한 보상을 지급할 수 있도록 함

---

### Track B: 영리 법인 (Pacemaker Inc.)

> 💵 "수강생/기업이 돈을 낸다"

| 항목          | 내용                                                                                                |
| ------------- | --------------------------------------------------------------------------------------------------- |
| 상품명 (예시) | "Premium Portfolio & Career Jump-up Masterclass"                                                    |
| 타겟          | 정부 지원 대상이 아닌 유학생 / 이미 취업해 있지만 이직을 원하는 경력자 / 더 밀착 케어를 원하는 사람 |
| 내용          | 비영리 프로그램보다 더 심화된 내용 / 1:1 코칭 추가, 연봉 협상 전략 등 '프리미엄' 서비스 제공        |
| 수익 구조     | 수강생에게 직접 수업료(Tuition)를 받음                                                              |

---

## 💡 현재 상황과 전략적 방향

### ✅ 보유 자산

- 현재 수강생 반응이 좋은 커리어 코칭, 6~8주 멘토링 프로그램
- 강사진 풀 보유
- paceupcareer.com 플랫폼 개발 중

### ⏰ 현실적 제약

- 온라인 강의 제작에 들일 시간과 에너지 부족

---

## 🚀 전략의 핵심: "라이브(Live)를 팔고, 녹화본(Archive)을 쌓는다"

강의를 미리 제작하지 않고, 현재 운영 중인 6주/8주 라이브 멘토링 코스를 'Paceup' 플랫폼 위에서 기수제로 진행하며 콘텐츠를 확보합니다.

### A. Paceup 플랫폼의 역할 변경 (VOD → 기수제 운영)

현재 개발 중인 사이트에서 '동영상 강의 재생' 기능보다는 '기수 관리(Cohort Management)' 기능을 우선적으로 활용

| 기능         | 설명                                                                                            |
| ------------ | ----------------------------------------------------------------------------------------------- |
| 모집 및 결제 | 6주 디자인 코스, 8주 개발 코스 신청을 Paceup 사이트에서 직접 받음                               |
| 자료실       | 멘토들이 나눠주는 PDF, 과제, 공지사항을 카카오톡 대신 Paceup 사이트 내 '강의실' 게시판에 업로드 |
| 결과         | 겉보기에는 온라인 교육 사이트의 구색을 갖추게 되며, 유저 트래픽이 자연스럽게 모이기 시작        |

### B. 콘텐츠 확보 (One Source Multi Use)

라이브 수업을 통해 콘텐츠를 자동으로 축적하여 미래 자산으로 만듭니다.

```
[Zoom 라이브 수업] → [녹화] → [Paceup 사이트에 업로드 (수강생 전용 복습 영상)]
                                         ↓
                              [기수 반복 시 아카이브 축적]
                                         ↓
                     [미래: 반응 좋은 영상 → 유료 VOD로 전환]
```

📌 참고: 펀딩을 받을 경우 컨텐츠 사용권한 확인 필요

## 📋 현실적으로 적용 가능한 실행 전략

### Step 1: [비영리] 8주 이력서/인터뷰 과정 (The Entry Point)

| 항목               | 내용                                                                                                         |
| ------------------ | ------------------------------------------------------------------------------------------------------------ |
| 운영 주체          | 100% 비영리 단체 (Paceup Career Society) 명의로 운영                                                         |
| 대상               | 취업을 희망하는 이민자 및 청년층 모두                                                                        |
| 비용               | 무료 (정부 및 YMCA 펀딩으로 운영)                                                                            |
| 내용               | 이력서 작성, 링크드인 관리, 일반적인 인사(HR) 인터뷰 대비, 캐나다 직장 문화 등 '보편적 취업 소양' 교육       |
| Paceup 플랫폼 활용 | 오프라인 수업 진행 시에도, 과제 제출(이력서 첨삭 요청) 및 공지사항 확인은 반드시 Paceup 웹사이트를 통해 진행 |

💡 기대 효과:

- 사이트 가입자 수 폭발적 증가
- 정부 제출용 교육 관리 데이터(LMS 로그) 자동 축적
- ⚠️ 자료 녹화 필수

---

### Step 2: [영리] 직무별 심화 포트폴리오/기술 멘토링 (The Premium)

| 항목      | 내용                            |
| --------- | ------------------------------- |
| 운영 주체 | 영리 법인 (Pacemaker)           |
| 대상      | Step 1 수료생 및 외부 유입 인원 |
| 비용      | 유료 (수익 창출 목적)           |

#### 프로그램 예시

| 직무        | 프로그램명                                      |
| ----------- | ----------------------------------------------- |
| 🎨 디자이너 | "디렉터와 함께하는 6주 포트폴리오 완성반"       |
| 💻 개발자   | "시니어 개발자의 코드 리뷰 및 기술 면접 대비반" |

#### 제안 논리

> "Step 1을 통해 이력서 준비는 완벽히 완료되었을 것입니다. 이제 서류 통과 후 '실력'을 입증할 포트폴리오가 필요합니다. 본 과정은 현직 디렉터의 1:1 맞춤 지도가 필요하므로 유료 과정으로 진행됩니다."
